# BFME II Archive

BFME II Archive is a small fan archive for custom maps and related files for *The Battle for Middle-earth II* and *The Rise of the Witch-king*.

The website opens with a Doors of Durin themed entrance screen and leads to a map archive with previews, search, sorting, filtering, and downloads. Additional non-map files are collected separately.

## Project Structure

- `index.html` - start page
- `maps.html` - map archive
- `other.html` - additional downloads
- `about.html` - project information
- `imprint.html` - imprint information
- `index.css` - start page styling
- `pages.css` - content page styling
- `app.js` - interactive behavior
- `data/` - images, audio, and downloads
