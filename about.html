About This Project

This website was created as a small archive for custom maps and useful files for The Battle for Middle-earth II and The Rise of the Witch-king.

The main focus is collecting maps in one simple place, so friends and players can easily browse, preview, and download them.

Additional files such as the RotWK Trainer are placed in the Other section.

This is a fan project created for personal use and for a university seminar about learning how to build websites.

BFME II Archive fan project for educational purposes.
